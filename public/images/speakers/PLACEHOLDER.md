# Dossier des photos d'orateurs

Placez vos photos d'orateurs ici.

## Format recommandé :
- Images JPG ou PNG
- Résolution : 200x200 pixels minimum
- Nom du fichier : utilisez le nom de l'orateur (ex: john-doe.jpg)

## Comment ajouter :
1. Exportez vos données JSON depuis l'app (Paramètres > Données)
2. Extrayez les photos du fichier JSON
3. Copiez vos images dans ce dossier
4. Mettez à jour les chemins dans le code
