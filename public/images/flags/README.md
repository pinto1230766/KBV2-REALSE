# Dossier des drapeaux

Placez les images de drapeaux ici.

Formats supportés : PNG, JPG, SVG
Tailles recommandées : 64x64 pixels

Noms des fichiers :
- france.png ou france.jpg (pour le français)
- portugal.png ou portugal.jpg (pour le portugais)
- cape_verde.png ou cape_verde.jpg (pour le créole cap-verdien)
